ListenToThis
Nodejs Angular Spotify API

ListenToThis is a music sharing service that allows you to visualize playlists in an appealing manner. Utilizing the Spotify and Echonest API , users can share their playlists and allow others to listen to the music while they explore the playlist. Using SPSSs (simple perfect squared squares) music is arranged into a wall cover that can be explored for artist bio, upcoming tours, and recent news about the artist.
